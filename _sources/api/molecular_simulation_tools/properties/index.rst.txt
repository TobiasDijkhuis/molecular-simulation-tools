.. py:function:: calculate_radial_distribution_function(images: list[ase.Atoms], elements: str | int | list | tuple | None = None, rmax: float = 6, nbins: int = 200) -> tuple[numpy.ndarray, numpy.ndarray, numpy.ndarray]

   Calculate the radial distribution function (RDF).

   :param images: List of frames
   :type images: list[Atoms]
   :param elements: List of elements to calculate the RDF for. If None, calculate it for all
                    elements. Default = None.
   :type elements: str | int | list | tuple | None
   :param rmax: Mamximum distance to calculate the RDF for in Angstrom. Default = 6.
   :type rmax: float
   :param nbins: number of bins to calculate between 0 and `rmax`. Default = 200.
   :type nbins: int

   :returns: * **distances** (*numpy.ndarray*) -- Array of distances
             * **average_rdf** (*numpy.ndarray*) -- Average RDF over all frames at each distance
             * **std_rdf** (*numpy.ndarray*) -- Standard deviation of the RDF over all frames at each distance

   :raises RuntimeError: If something went wrong with the distances.

.. py:function:: get_dipole_moment(positions: numpy.ndarray, charges: numpy.ndarray) -> numpy.ndarray

   Get the dipole moment of a set of point charges.

   :param positions: Positions of point charges. NxM array
   :type positions: numpy.ndarray
   :param charges: Charges. Array of length N
   :type charges: numpy.ndarray

   :returns: Dipole moment. Array of length M
   :rtype: numpy.ndarray

   :raises ValueError: If the shapes of `positions` and `charges` do not match.
   :raises ValueError: If the sum of all `charges` is not 0 (i.e. the structure is not neutral).


.. py:function:: get_h2o_charges(symbols: list[str]) -> numpy.ndarray

   Assign charges of H2O.

   :param symbols: List of symbols (i.e. "H" or "O")
   :type symbols: list[str]

   :returns: **charges** -- List of charges corresponding to the atoms in `symbols`.
   :rtype: numpy.ndarray

.. py:function:: get_ir_spectrum(frames: list[ase.Atoms], charges: numpy.ndarray, timestep: float, step: int = 1, pad: bool = True, window_type: str | None = None) -> tuple[numpy.ndarray, numpy.ndarray]

   Calculate the infrared spectrum of an MD trajectory.

   :param frames: frames of molecular dynamics simulations
   :type frames: list[Atoms]
   :param charges: charge of each atom in the Atoms
   :type charges: numpy.ndarray
   :param timestep: timestep between frames in seconds
   :type timestep: float
   :param step: Number of steps to take between frames. Default: 1
   :type step: int
   :param pad: Whether to pad the DACF with zeros to have the FFT be faster.
               See https://docs.scipy.org/doc/scipy/reference/generated/scipy.fft.next_fast_len.html.
               Default: True
   :type pad: bool
   :param window_type: Window type to apply to get rid of finite
                       sampling effects. See https://docs.scipy.org/doc/scipy/reference/generated/scipy.signal.windows.get_window.html#scipy.signal.windows.get_window
                       for options. Default: None
   :type window_type: str | None

   :returns: * **xf** (*numpy.ndarray*) -- frequencies in Hz
             * **spectrum** (*numpy.ndarray*) -- intensity at each frequency. Arbitrary units
