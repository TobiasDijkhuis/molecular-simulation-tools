.. py:function:: convert_cartesian_to_spherical(position: numpy.ndarray) -> tuple[float, float, float]

   Convert cartesian coordinates to spherical coordinates.

   See https://en.wikipedia.org/wiki/Spherical_coordinate_system#Cartesian_coordinates

   :param position: Cartesian coordinate vector
   :type position: numpy.ndarray

   :returns: Tuple containing (r, theta, phi)
   :rtype: tuple[float, float, float]

   :raises ValueError: If `position` is not a 3D vector, i.e. does not have shape ``(3,)``.


.. py:function:: convert_spherical_to_cartesian(r: float, theta: float, phi: float) -> numpy.ndarray

   Convert spherical coordinates to cartesian coordinates.

   See https://en.wikipedia.org/wiki/Spherical_coordinate_system#Cartesian_coordinates

   :param r: Radial distance
   :type r: float
   :param theta: Polar angle in radians
   :type theta: float
   :param phi: Azimuthal angle in radians
   :type phi: float

   :returns: numpy array containing (x, y, z)
   :rtype: numpy.ndarray


.. py:function:: correct_distance_for_pbc(distance: numpy.ndarray, box_length: float) -> numpy.ndarray

   Correct a distance for periodic boundary conditions.

   :param distance: Array of distances
   :type distance: numpy.ndarray
   :param box_length: Length of the periodic box along the dimension of `distance`
   :type box_length: float

   :returns: **distance** -- Corrected distance
   :rtype: numpy.ndarray


.. py:function:: get_random_unit_vector(n: int = 1) -> numpy.ndarray

   Get a random vector on the unit sphere.

   Uses method described in https://mathworld.wolfram.com/SpherePointPicking.html

   :param n: Number of vectors to generate. Default = 1
   :type n: int

   :returns: **vectors** -- Numpy matrix of shape Nx3, or just vector of shape (3,) if
             `n` is 1.
   :rtype: numpy.ndarray


.. py:function:: project_on_unit_sphere(vector: numpy.ndarray) -> numpy.ndarray

   Project a vector on the unit sphere, by dividing it by its length.

   :param vector: N-dimensional vector
   :type vector: numpy.ndarray

   :returns: Vector projected on N-dimensional unit sphere
   :rtype: numpy.ndarray


.. py:function:: sample_random_rotation() -> tuple[numpy.ndarray, float]

   Get a random rotation axis and rotation angle.

   Uses method described in https://math.stackexchange.com/questions/442418/random-generation-of-rotation-matrices

   :returns: * **rotation_axis** (*numpy.ndarray*) -- 3-dimensional rotation axis.
             * **rotation_angle** (*float*) -- Rotation angle in radians.


.. py:function:: turn_grid_into_position_vectors(grid_matrices: tuple[numpy.ndarray, Ellipsis]) -> numpy.ndarray

   Turn a grid created by :func:`numpy.meshgrid` into position vectors.

   Taken from https://stackoverflow.com/questions/12864445/how-to-convert-the-output-of-meshgrid-to-the-corresponding-array-of-points

   :param grid_matrices: Tuple of N grid matrices with M points.
   :type grid_matrices: tuple[numpy.ndarray, ...]

   :returns: MxN numpy array of the grid positions in N dimensions.
   :rtype: numpy.ndarray
