.. py:function:: calculate_rmsd(atoms: ase.Atoms, target: ase.Atoms, indices: list[int] | None, permute: bool = True, return_permuted_target: bool = True) -> tuple[float, tuple[ase.Atoms, ase.Atoms]]

   Calculate the root-mean-squared-displacement (rmsd) between atoms `atoms` and `target`.

   :param atoms: atoms
   :type atoms: Atoms
   :param target: target
   :type target: Atoms
   :param indices: List of indices, or None if all atoms.
   :type indices: list[int] | None
   :param permute: Whether to permute the atom indices to get the minimum
                   rmsd. Only permutes between identical elements. Default = True.
   :type permute: bool
   :param return_permuted_target: Whether to return the permuted target,
                                  that has the minimum rmsd. Default = True.
   :type return_permuted_target: bool

   :returns: * **min_rmsd** (*float*) -- minimum rmsd
             * **return_atoms** (*tuple[Atoms, Atoms]*) -- tuple of `atoms` and `target` with only the indices in `indices`.
               If `return_permuted_target` is True, returns the permuted `target` that leads
               to the minimum rmsd.

   :raises ValueError: If the number of atoms, order of elements or cells are different between
       `atoms` and `target`.

.. py:function:: find_min_height_for_adsorbate_on_surface(surface: ase.Atoms, ngrid: int | tuple[int, int], distance: float, adsorbate: ase.Atoms | None = None) -> tuple[numpy.ndarray, numpy.ndarray, numpy.ndarray]

   Find the minimum height to place an adsorbate at least `distance` away from surface atoms.

   Calculate the minimum height for `adsorbate` for it to have all its
   atoms at least `distance` away from the other atoms in `surface`.

   The returned points correspond to placing the center-of-mass of `adsorbate`
   at ``(x, y, z)``, such that the minimum distance to any atom in the surface is
   `distance`.

   :param surface: Surface
   :type surface: Atoms
   :param ngrid: Number of grid points along x and y
   :type ngrid: int | tuple[int, int]
   :param distance: Distance from all other atoms.
   :type distance: float
   :param adsorbate: Adsorbate. If None, create a dummy Atoms instance with one atom.
                     Default = None.
   :type adsorbate: Atoms | None

   :returns: * **grid_x** (*np.ndarray*) -- grid of positions along x
             * **grid_y** (*np.ndarray*) -- grid of positions along y
             * **sample_heights** (*np.ndarray*) -- height of the points corresponding to the points in `grid_x` and `grid_y.


.. py:function:: find_min_height_for_distance(x: float, y: float, point_coordinates: numpy.ndarray, distance: float, box_size: numpy.ndarray | None = None) -> float

   Find the minimum height for a point to be `distance` away from other points.

   Calculate the minimum height ``z`` for a point ``(x, y)`` for it to be at least
   `distance` away from all points in `point_coordinates`.

   :param x: x-coordinate of desired point
   :type x: float
   :param y: y-coordinate of desired point
   :type y: float
   :param point_coordinates: Nx2 (or more) array of N point coordinates.
   :type point_coordinates: np.ndarray
   :param distance: Minimum distance to all other points.
   :type distance: float
   :param box_size: Size of the box. If None, do not include periodic boundary conditions.
                    Default = None.
   :type box_size: np.ndarray | None

   :returns: **height** -- Minimum height required for ``(x, y)`` to be `distance` away from all other
             points.
   :rtype: float

   :raises ValueError: If no points within radius `distance` from ``(x, y)``
       are found in `point_coordinates` (in a 2D projection).

.. py:function:: sample_new_point(points: numpy.ndarray, minimum_distance: float, initial_spawn_distance: float | None = None, rtol: float = 1e-05, atol: float = 1e-08, n: int = 1) -> numpy.ndarray

   Generate a new point in a random direction that is a certain distance from other points.

   :param points: NxM array of coordinates of all old points,
                  where N is the number of points and M the dimension
   :type points: np.ndarray
   :param minimum_distance: Minimum distance for the newly generated
                            point to be from all other points
   :type minimum_distance: float
   :param initial_spawn_distance: Initial distance from origin.
                                  If None, is the same as minimum_distance. Default = None
   :type initial_spawn_distance: float | None
   :param rtol: Relative tolerance. Default: 1e-5
   :type rtol: float
   :param atol: Absolute tolerance. Default: 1e-8
   :type atol: float
   :param n: number of points to generate. Default: 1
   :type n: int

   :returns: **r_vec** -- Array of length M of coordinates of new point.
             Shape n*M if multiple points are being generated.
   :rtype: np.ndarray

   :raises ValueError: If `minimum_distance` or `initial_spawn_distance` is less or equal to 0
