Identify molecules in a structure based on the connected subgraphs.

Requires networkx to be installed.

Parameters:

  • atoms (Atoms) – Atoms instance to identify molecules in

  • cutoffs (dict[str, float] | None) – cutoffs of each element. Dictionary with keys for the symbols and values of the cutoff radii. If None, use the ase.data.covalent_radii. Default: None

Returns:

c_list – Array of lists of connected atom indices
