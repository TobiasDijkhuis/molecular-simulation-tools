molecular_simulation_tools.properties.calculate_radial_distribution_function(images: list[ase.Atoms], elements: str | int | list | tuple | None = None, rmax: float = 6, nbins: int = 200) → tuple[numpy.ndarray, numpy.ndarray, numpy.ndarray][source]

Calculate the radial distribution function (RDF).

Parameters:

  • images (list[Atoms]) – List of frames

  • elements (str | int | list | tuple | None) – List of elements to calculate the RDF for. If None, calculate it for all elements. Default = None.

  • rmax (float) – Mamximum distance to calculate the RDF for in Angstrom. Default = 6.

  • nbins (int) – number of bins to calculate between 0 and rmax. Default = 200.

Returns:

  • distances (numpy.ndarray) – Array of distances

  • average_rdf (numpy.ndarray) – Average RDF over all frames at each distance

  • std_rdf (numpy.ndarray) – Standard deviation of the RDF over all frames at each distance

Raises:

RuntimeError – If something went wrong with the distances.

molecular_simulation_tools.properties.get_ir_spectrum(frames: list[ase.Atoms], charges: numpy.ndarray, timestep: float, step: int = 1, pad: bool = True, window_type: str | None = None) → tuple[numpy.ndarray, numpy.ndarray][source]

Calculate the infrared spectrum of an MD trajectory.

Parameters:

  • frames (list[Atoms]) – frames of molecular dynamics simulations

  • charges (numpy.ndarray) – charge of each atom in the Atoms

  • timestep (float) – timestep between frames in seconds

  • step (int) – Number of steps to take between frames. Default: 1

  • pad (bool) – Whether to pad the DACF with zeros to have the FFT be faster. See https://docs.scipy.org/doc/scipy/reference/generated/scipy.fft.next_fast_len.html. Default: True

  • window_type (str | None) – Window type to apply to get rid of finite sampling effects. See https://docs.scipy.org/doc/scipy/reference/generated/scipy.signal.windows.get_window.html#scipy.signal.windows.get_window for options. Default: None

Returns:

  • xf (numpy.ndarray) – frequencies in Hz

  • spectrum (numpy.ndarray) – intensity at each frequency. Arbitrary units
